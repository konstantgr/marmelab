Описание функционала

Set current coordinates текущие координаты сканера подставляются в таблицу в строку 'Begin coordinates' Points/Step выбор разбиения пути: на определенное число точек или разбиение с заданным шагом Shake/Lines выбор траектории обхода точек: змейкой или по линиям Relative coordinates задает систему отсчета относительно текущего положения сканера. То есть делает нули строки 'Begin coordinates' в той точке, где находится сканер в данный момент. 'End coordinates' будут выставлены относительно текущего положения сканера Важно! Порядок, в котором будут проводиться измерения вдоль осей, выставляется в строке 'Order'. Порядок '0' соответствует первому, '1' - второму и тд.. Строка 'Status' позволяет отключить сканирование по соответствующей оси. Значение '1' включает ось, значение '0' выключает
